# Частица 的 (de)

Структурная частица **的** (de) — одна из самых употребительных в китайском языке.

## Основные функции

### 1. Притяжательная конструкция

Частица **的** связывает владельца с объектом владения:

```
我的书 — моя книга
他的朋友 — его друг
中国的文化 — китайская культура
```

### 2. Определительная конструкция

Прилагательное + **的** + существительное:

```
好的学生 — хороший студент
漂亮的花 — красивые цветы
```

### 3. Номинализация

**的** может превращать прилагательное или глагольную группу в существительное:

```
学中文的 — тот, кто учит китайский
我买的书 — книга, которую я купил
```

## Примеры предложений

| Китайский | Русский |
|-----------|---------|
| 这是我的书。 | Это моя книга. |
| 他是好的老师。 | Он хороший учитель. |
| 我喜欢红色的。 | Мне нравится красный (цвет). |

## Когда можно опустить 的

Между местоимением и существительным, обозначающим близкого человека или принадлежность к группе, **的** часто опускается:

```
我妈妈 (а не 我的妈妈) — моя мама
我们学校 — наша школа
```
